Debug the API Response

You are building a pet adoption app. The app made a request to an API, but the response is broken. Your job is to fix the JSON so the app can show the pets correctly.

This activity helps students practice reading JSON, noticing data type problems, and debugging data the way a developer would in a real workflow.

What the app expects

Each pet in the pets array should have these fields:

  • id: number
  • name: string
  • type: string
  • age: number
  • vaccinated: boolean
  • adopted: boolean

API request

GET /api/pets

Your fix

Fix the JSON so each pet matches the expected schema, then click "Validate JSON".

Nice work. Your JSON is valid and matches the schema, so the app can use it.

App preview

When your data is fixed, the pet cards will appear here.

Teacher idea: after students succeed, ask which bug they noticed first and which one was easiest to miss.
