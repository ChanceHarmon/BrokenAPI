# API Debugger

A small interactive web app where students debug a broken API response by fixing JSON so the app can render pet adoption cards correctly.

## Live Demo

[View the live project](https://chanceharmon.github.io/BrokenAPI/)

## Built With

- HTML
- CSS
- JavaScript

## Features

- Loads a broken JSON API response
- Lets users edit the JSON directly
- Validates the response against an expected schema
- Shows hints for common debugging issues
- Renders pet cards after the data is fixed

## Purpose

This project was designed as a simple interactive debugging activity for learners. It gives students practice with:
- reading JSON
- spotting schema mismatches
- identifying incorrect data types
- fixing realistic API response issues

## Run Locally

Clone the repo and open `index.html`, or run it with Live Server in VS Code.
